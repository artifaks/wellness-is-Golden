A comprehensive herbal medicine database and search application.

== Description ==
Wellness is Golden provides a searchable database of herbs and their medicinal properties. This plugin allows you to:

* Create and manage a database of medicinal herbs
* Add scientific names and dosage information
* Categorize herbs by type and properties
* Provide a searchable interface for users
* Display detailed herb information

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/wellness-is-golden`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode [wellness_golden] to display the herb search interface
4. Begin adding herbs through the admin interface

== Frequently Asked Questions ==
= How do I add new herbs? =
Go to the Herbs menu in your WordPress admin and click "Add New"

= How do I display the search interface? =
Use the shortcode [wellness_golden] in any post or page
